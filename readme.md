## Запуск
### web-ui
- `./gradlew web-ui:shadowJar -DORG_GRADLE_PROJECT_isProduction=true` - собрать `JAR` файл со всеми необходимыми зависимостями
- `java -jar web-ui/build/libs/web-ui-0.0.1-all.jar` - запуск приложения
